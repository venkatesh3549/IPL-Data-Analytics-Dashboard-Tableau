# IPL Data Analytics Dashboard | Tableau

## Project Overview
This project presents an interactive Tableau dashboard built using IPL ball-by-ball data from 2008–2025. The dashboard provides comprehensive insights into team performance, player statistics, venue analysis, and season-wise trends.

## Key Features
- Total Runs and Total Matches KPI Cards
- Top Run Scorers Analysis
- Top Bowlers Analysis
- Team Performance Comparison
- Venue-wise Match Analysis
- Season-wise Run Trends
- Interactive Filters for Team, Venue, City, and Season

## Tools Used
- Tableau
- Microsoft Excel
- Data Analytics
- Data Visualization

## Dataset
IPL Ball-by-Ball Dataset (2008–2025)

## Insights Generated
- Identified leading run scorers and wicket-takers.
- Analyzed team performance across IPL seasons.
- Examined venue popularity and match distribution.
- Visualized season-wise scoring trends.
